The FourD Library

The FourD library allows web developers to visualize graphs in 3D. Desisgned from the get go to run in the browser, it's been tested in Chrome, Firefox, and Internet Explorer 11 (scary settings permitting ;). Rumor has it the library even works on a Raspberry PI.
